# Contributing to PartyShare

## Development setup
1. Install dependencies: `yarn install`
2. Start development server: `yarn dev`
3. Run checks before opening a PR:
   - `yarn format-ci`
   - `yarn test-ci`

## Pull request expectations
- Keep PRs focused and easy to review.
- Use clear commit messages.
- Update docs for workflow, architecture, or policy changes.

## Coding conventions
- Follow repository AGENTS guidance and existing Sharetribe template patterns.
- Use React Intl for user-facing strings.
